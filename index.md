## Building VKAX

```
git clone https://github.com/vkaxcore/VKAX &&
cd VKAX/depends/ &&
chmod +x conf* &&
make &&
cd .. &&
./autogen.sh &&
./configure --prefix=$PWD/depends/x86_64-pc-linux-gnu/ &&
make
```
